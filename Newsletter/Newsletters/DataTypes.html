Ruby essentials

Data Types

Numbers

Series of digits, using a dot as a decimal mark, like integers or float


# float type
distance = 0.1 
# integer and float type
time = 9.87 / 3600
speed = distance / time 
puts "The average speed is #{speed} km/h"
                        

>> The average speed is 36.4741 km/h

BOOLEANS

Represents only one bit of information either true or false


is_running = false
if is_running
  puts “System running“
else
  puts “System down“ 
end
                        

>> System down

STRINGS

Group of letters that represent a sentence or a word


puts "String Data Type";
puts 'escape using "\\"';
puts 'That\'s right';
                        

>> String Data Type
>> escape using "\"
>> That's right

HASHES

A hash assign its values to its key


colors = {
  "red" => 0xf00,
  "green" => 0x0f0,
  "blue" => 0x00f
}
colors.each do |key, value|
  puts key, " is ", value, "\n"
end 
                        

>> System down

ARRAYS

Stores data or list of data and It can contain all types of data


my_array = [10, 3.14, "lastElement"]
my_array.each do |i|
  puts i
end
                        

>> 10
>> 3.14
>> lastElement

SYMBOLS

It is preceded by a colon (:). They are used instead of strings because they can take up much less memory


my_domains = {
  :amazon => "amazon.com",
  :veeqo => "veeqo.com"
}
puts domains[:veeqo]
