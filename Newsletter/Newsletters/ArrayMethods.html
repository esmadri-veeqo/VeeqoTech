Common Array Methods

.first & .last

Accesses the first/last element of the array


array = [0, 1, 2, 3, 4]
array.first
array.last
                        

>> 0
>> 4

.push & .unshift

.push adds an element to the end and .unshift allow you to add an element to the beginning of an array


array = [0, 1, 2, 3, 4]
array.push(99)
array.unshift(-1)
                        

=> [0, 1, 2, 3, 4, 99]
=> [-1, 0, 1, 2, 3, 4, 99]

.select

Iterates over an array and returns a new array that includes any items that return true to the expression provided


array = [0, 1, 2, 3, 4]
array.select { |number| number > 3 }
                        

=> [4]

.include?

Checks to see if the argument given is included in the array


array = [1, 2, 3, 4, 5]
array.include?(3)
                        

>> true

.each

Iterates over each element of the array, allowing you to perform actions on them


array = [1, 2, 3]
array.each do |element|
  puts element
end
                        

>> 1
>> 2
>> 3

.map

The .map and .collect methods iterate over each element of the array, allowing you to perform actions on them


array = [1, 2, 3]
array.map { |element| element * 2 }
  puts element
end
                        

>> 2
>> 4
>> 6

.uniq

Takes in an array containing duplicate elements, and returns a copy of the array containing only unique elements


array = [1, 1, 2, 2, 3, 3, 3, 4, 4, 4, 4, 5, 6, 7, 8]
array.uniq
                        

>> [1, 2, 3, 4, 5, 6, 7, 8]

.sample

Return a random element or n random elements from the array


array = [18, 22, 33, nil, 5, 6] 
array.sample(2)
                        

>> [22, nil]
