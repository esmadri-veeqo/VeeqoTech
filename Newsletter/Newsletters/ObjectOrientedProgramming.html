What is object-oriented programming or OOP?

It's a programming model that organizes software design around data, or objects, rather than functions and logic.

Structure of object-oriented programming

Classes

User-defined data types that act as a shell or structure to create new objects, with specific attributes and methods


class Customer
   def initialize(id, name, addr)
      @cust_id = id
      @cust_name = name
      @cust_addr = addr
   end
end

Objects

Are instances created using a class with specifically defined data.


cust1 = Customer.new("1", "John", "Wisdom Apartments, Ludhiya")

cust2 = Customer.new("2", "Poul", "New Empire road, Khandala")

Methods

Are functions that are defined inside a class that describe the behaviors of an object.


class Customer
    def hello
        puts "Hello Ruby!"
    end
end

cust1 = Customer.new()
cust1.hello
                                

>> Hello Ruby!

Attributes

Objects will have data stored in the attributes field. Like in the previous example, we added the next attribute to the Customer class


@cust_id = id
@cust_name = name
@cust_addr = addr
